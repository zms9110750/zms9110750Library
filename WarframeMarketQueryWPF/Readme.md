<br/>输入"充沛"，会搜索从第一节以充沛开头的单词，例如"充沛射击"
<br/>分隔符表示跳过这一节的剩余内容。例如"w 一套"可以匹配"wukong prime 一套"和"wisp prime 一套"
<br/>
<br/>并且分隔符可以表示数量大于一的任节。例如" 一套"，只需要一个空格，就可以匹配"wisp prime 一套"
